# Zomoto_restaurant_analysis

## Project Overview
This project analyzes restaurant and order data using SQL.

## Features
- Top rated restaurants
- Revenue analysis
- Peak order hours
- Window functions
- Ranking system
- GROUP BY and HAVING
- Joins and subqueries

## Technologies Used

- SQL
- MySQL Workbench

## SQL Concepts Used

- GROUP BY
- HAVING
- JOIN
- Subqueries
- RANK()

- Views
